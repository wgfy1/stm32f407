# F407project

STM32F407 项目（基于 STM32CubeMX + HAL + LVGL）。

说明：
- 使用 CMake 构建。请在本地安装 `arm-none-eabi-gcc` 工具链。

将仓库推送到 GitHub 的建议步骤：
1. 在 GitHub 上创建一个空仓库（不要初始化 README）。
2. 在本地仓库中添加远程并推送：

```powershell
git remote add origin <YOUR_GITHUB_REPO_URL>
git push -u origin main
```

如果你希望我代为创建远程仓库并推送，请选择使用 `gh`（GitHub CLI）。
